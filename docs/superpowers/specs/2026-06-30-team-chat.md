# Team Chat & Activity Feed

## Overview

Team chat for studio members to communicate and see automatic activity updates (logo changes, game creation, trailer uploads, etc.) in a unified feed.

## Backend

### New model

```prisma
model StudioChatMessage {
  id        String   @id @default(cuid())
  studioId  String
  authorId  String
  message   String   @db.VarChar(2000)
  createdAt DateTime @default(now())

  studio Studio @relation(fields: [studioId], references: [id], onDelete: Cascade)
  author User   @relation(fields: [authorId], references: [id])

  @@index([studioId, createdAt])
  @@map("studio_chat_messages")
}
```

### Endpoints

| Method | Path | Auth | Description |
|---|---|---|---|
| `GET` | `/studios/:slug/activities` | Studio member | Combined feed (chat + audit logs) |
| `POST` | `/studios/:slug/chat` | Studio member | Post a chat message |

### Combined feed

`GET /studios/:slug/activities` returns:
```json
{
  "items": [
    { "type": "chat", "id": "...", "author": {...}, "message": "...", "createdAt": "..." },
    { "type": "system", "id": "...", "action": "LOGO_CHANGED", "actor": {...}, "createdAt": "..." }
  ]
}
```

## Frontend

- Chat input at the bottom of the team page
- Message bubbles with avatar, name, timestamp
- System events shown with a 🤖 icon
- Auto-scroll to bottom on new messages
